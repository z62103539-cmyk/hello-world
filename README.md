Android App Project Outline
Project Name: Simple Note Pad

1. Project Description
Simple Note Pad is a basic Android application that allows users to create, view, and delete short text notes. It serves as a personal digital notebook for quick thoughts and daily reminders.

2. Problem Addressing
People often forget small details or ideas throughout the day. This app provides a clean, distraction-free tool to quickly jot down information without the complexity of heavy office software.

3. Platform
Operating System: Android
Development Tool: Android Studio
Language: Java / Kotlin Beginner Level

4. Front/Back end Support
Frontend: Standard XML Layouts using UI components like EditText, Button, and RecyclerView.
Backend: Local data storage using SharedPreferences or a simple SQLite database as covered in the beginner curriculum.

5. Functionality
Add Note: A text field to type in and a button to save the note.
View Notes: A simple list view to display all saved notes.
Delete Note: Ability to remove a note once it is no longer needed.

6. Design Wireframes
The app will feature a two-screen design:
Main Screen: Displays the list of notes with a Floating Action Button to add a new one.
Input Screen: A simple page with a title bar, a large text area for typing, and a Save button.
